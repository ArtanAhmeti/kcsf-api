API for KCSF Questionnaire Answers

An API to access data used by the KCSF Visualizer Index 2016.

Description:

From requests in this API, you will get comparison info reponse(JSON) according to the questions(q1_id, q2_id) and language(lang) that you will pass in this API request.

Guide

URL:

/comparison

Method:

POST

Data Params:

        
          {
          	"q1_id": [alphanumeric],
          	"q2_id": [alphanumeric],
          	"lang": [alphanumeric]
          }
Note! The values of q1_id and q2_id should be in this form "q14", the "q" is for question and 14 is sample number of question, please refer to this document to see all questions of this questionnaire Download questionnaire. And the the values of "lang" can be "sq", "en" and "sr".
Note! The API request must be sent as POST request with application/json type in body.

Success Sample Request

URL:

/comparison

Data Params Values:

        
          {
            "q1_id": "q122",
          	"q2_id": "q14",
          	"lang": "en"
          }
